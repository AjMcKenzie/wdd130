Playstation 5




Uncharted Drake's Fortune

    This is the first game of the "Uncharted" series. This game was a fun and it wasn't a difficult game, but it did have some parts that were a little more challenging. Not only did this game have some good mechanics, but the story was amazing. This being an older game the graphics aren't as good as the newer games, that being said it still looks good. The story makes it worth playing even with graphics not being as good.
    The game has puzzles, parkour, and of course some shooting to escape. This game is all about unearthing treasure and beating the antagonist to the treasure. As you race to the treasure you can find smaller treasure around the game. The game is pretty linear, but a lot of game are especially older games before “open world” was a more common thing.




Pros

  • Has a great story
  • Fun to play
  • Has a variety of different mechanics

Cons

  • Has some glitches
  • Graphics are a little rough
  • Linear gameplay
  • Playstation exclusive
